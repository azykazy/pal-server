# クラウドは Azure を採用 (AWS/GCP 比較)

## 状況

Palworld サーバー (4vCPU/16GB、東京近辺、スポット、使用時のみ課金) をどのクラウドに置くか。
ユーザーの事前調査では Azure 最安の結論で、その裏付けが必要だった。

## 決定

Azure (japaneast, Standard_D4as_v5 Spot) を採用する。

## 理由

2026-07 時点の公式価格での月30時間プレイ比較 (コンピューティング+ディスク64GB+IP):

| 構成 | Azure | AWS | GCP |
|---|---|---|---|
| スポット単価/時 | $0.0414 (D4as_v5) | 約$0.085 (m6a.xlarge) | $0.038〜0.079 (t2d/n2d) |
| ディスク64GB/月 | $4.80 | $6.14 | $8.32 |
| 月額合計 (固定IP) | 約$9.7 | 約$12.3 | 約$13.1 |
| 月額合計 (動的IP運用) | 約$6.2 (IP都度削除) | 約$8.8 | 約$9.6 |

- Spot 単価が AWS 東京の約半額、ディスク (Standard SSD) が3社最安
- IP を起動時作成・停止時削除する運用 ([[DEC-20260717-002]]) なら動的 IP 前提でも Azure が最安

## トレードオフ

- Azure Spot の eviction 通知は30秒 (AWS は2分)
- GCP t2d の瞬間スポット価格には負けることがあるが、変動が大きく安定比較では Azure 有利
