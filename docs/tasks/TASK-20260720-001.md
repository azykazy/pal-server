# コミュニティサーバーモード切り替え・パスワード12桁強化

## 実行内容

### 変更ファイル

1. **vm/docker-compose.yml.tftpl**
   - `COMMUNITY` を環境変数 `${community_mode}` に変更（Key Vault 管理）
   - `SERVER_NAME` を環境変数 `${server_name}` に変更（Key Vault 管理）

2. **vm/fetch-secrets.sh.tftpl**
   - Key Vault から `community-mode` と `server-name` を読み取り `.env` に出力
   - パスワード生成ロジックを 4 桁から 12 桁に強化（より強固なセキュリティ）
   - 既存シークレット取得処理は維持

3. **scripts/update-config.sh**
   - `community on|off` コマンド追加
     - Key Vault で `community-mode` キーを更新
     - VM 上の `docker-compose.yml` を `sed` で動的に書き換え
     - コンテナを再起動（`docker-compose up -d`）
   - パスワード生成ロジックを 4 桁から 12 桁に強化

### 実装パターン

```bash
# 使用方法
mise run update-config -- community on   # コミュニティモード有効化
mise run update-config -- community off  # コミュニティモード無効化
```
