# PUBLIC_IP を IMDS から取得しコミュニティサーバーリストへの表示を修正

## 実行内容

### 変更内容

1. **vm/fetch-secrets.sh.tftpl**
   - Azure IMDS（Instance Metadata Service）から公開 IP アドレスを取得する処理を追加
   - 取得した PUBLIC_IP を .env ファイルに追記

2. **vm/docker-compose.yml.tftpl**
   - `COMMUNITY` フラグを `false` から `true` に修正
   - `PUBLIC_IP` 環境変数を docker-compose に追加

3. **稼働中 VM への対応**
   - fetch-secrets.sh と docker-compose.yml を直接更新
   - サービス再起動対応済み

## 背景

コミュニティサーバーブラウザでこのサーバーが表示されないという問題の調査から発生したタスク。サーバーが公開 IP を認識していないことが原因だったため、IMDS から動的に取得するように改善。

## 結果

- PR #23 作成・マージ完了
- URL: https://github.com/azykazy/pal-server/pull/23
- コミュニティサーバーブラウザに正常に表示されるようになった
